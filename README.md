支付完成后，VPS 会自动开通，几分钟内你的注册邮箱会收到一封系统邮件，里面有 SSH 端口号——注意这个端口不是 22，搬瓦工用的是一个随机的高位端口。

## 搬瓦工搭建教程第二步：登录 KiwiVM 控制面板

KiwiVM 是搬瓦工自研的 VPS 管理后台，搭建过程中很多操作都在这里完成。

### 进入 KiwiVM 的步骤

1. 登录搬瓦工用户中心
2. 点击顶部菜单 **Services** → **My Services**
3. 找到你购买的服务，点右侧的 **Open KiwiVM** 按钮

进去之后你会看到 **Main Controls** 页面，这里有几个常用信息：

- **IP Address**：你的 VPS 公网 IPv4 地址
- **SSH Port**：SSH 登录端口（就是邮件里那个，通常是 5 位数字）
- **Root Password Reset**：重置 root 密码的入口（新买的 VPS 先在这里重置一次，获取初始密码）

顺便说一下 KiwiVM 里几个常用功能：

- **Install New OS**：重装系统，支持 Ubuntu、Debian、Rocky Linux 等十几个系统，装前记得先 Stop 关机
- **Migration**：机房迁移，CN2 GIA-E 套餐可以在这里一键免费换机房
- **Backups / Snapshots**：查看和恢复自动备份快照，每台机器可以保留两个快照
- **Statistics**：流量使用图表，看本月用了多少带宽

## 搬瓦工搭建教程第三步：SSH 连接到 VPS

有了 IP、端口、密码，下一步就是用 SSH 工具连上服务器。

### Windows 用户

推荐用 Xshell（官方有免费的家庭版）或者 PuTTY。

以 Xshell 为例：
1. 打开软件，新建会话
2. 填入 **Host**（服务器 IP）和 **Port**（KiwiVM 里看到的端口）
3. 点 Connect，验证方式选 Password
4. 用户名填 `root`，密码粘贴刚才重置的密码

密码输入时终端不显示任何字符，直接粘贴后按回车就行。出现 `root@bwg:~#` 或类似提示说明连接成功。

### Mac 用户

直接用系统自带终端：

bash
ssh root@你的IP -p 你的端口


比如 IP 是 `1.2.3.4`，端口是 `29378`，就输入：

bash
ssh root@1.2.3.4 -p 29378


回车后输入密码登录。

连进去之后建议做一件事：改 SSH 默认端口。不改的话 22 端口会被大量扫描，日志里全是暴力破解记录，时间长了有一定安全风险。

## 搬瓦工搭建教程第四步：安装系统和部署应用

SSH 连进去了，接下来就是你自己的服务器了。

### 先跑一个系统检测脚本（可选）

bash
wget -qO- bench.sh | bash


这个脚本会测 CPU、内存、硬盘 IO、网络带宽等，跑完你就知道买到的机器性能大概在什么水平。

### 常见搭建场景

**建站（WordPress + LNMP）**

一键包推荐 LNMP（Linux + Nginx + MySQL + PHP）：

1. 下载安装包后按提示选择组件版本
2. 安装完成后在 Nginx 里配置域名和虚拟主机
3. 下载 WordPress 程序，导入数据库，按向导完成安装

整个过程大概需要 30-60 分钟，主要在等 LNMP 编译。

**部署 Docker 应用**

搬瓦工 CN2 GIA-E 套餐最低 1GB 内存，跑单个轻量级 Docker 容器没问题。如果要跑 Dify、n8n 这类带 AI 功能的完整应用，建议至少选 2GB 内存版本（即 CN2 GIA-E 套餐 B 或以上）。

先装 Docker：

bash
curl -fsSL https://get.docker.com | bash


装完可以直接 `docker run` 或者用 `docker-compose` 编排。

**开启 BBR 加速**

BBR 是 Google 出的 TCP 拥塞控制算法，开了之后网络吞吐量和延迟表现都会有改善。搬瓦工部分系统默认已经开启，没开的话可以用一行命令检测：

bash
sysctl net.ipv4.tcp_congestion_control


输出 `bbr` 说明已经开了。没开的话网上有一键脚本，搜一下就有，这里不展开。

## 一些容易踩的坑

IP 被封是最常见的问题。搬瓦工分配的 IP 如果遭到防火墙封锁，需要通过 KiwiVM 迁机房来换 IP（迁完 IP 就变了），或者付费申请更换（约 $8/次）。实际用下来 CN2 GIA 机房的 IP 相对稳定一些，DC6 和 DC9 被封的频率比普通机房低。

账号注册不要用代理 IP，不然下单可能被系统判定为欺诈订单。如果购买时提示 Fraud，检查一下本地 IP 是否干净。

流量超额不会直接断线，而是限速到很低，月底会自动恢复。买之前估算好用量，1TB 月流量日常建站或轻量应用绝对够，高流量场景考虑 2TB 版本。
